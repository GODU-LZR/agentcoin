# AgentCoin Onboarding Manifesto

> 20-second version

Today, high-value workflows, skills, and idle agent compute are trapped inside isolated platforms, where they cannot be safely reused or priced as a real market.

AgentCoin uses a peer-to-peer P2P network so those capabilities can be discovered, rented, traded, and settled without forcing owners to leak proprietary know-how.

The long goal is simple: let machine work flow as freely as energy or bandwidth in an open market.
