# AgentCoin Onboarding Manifesto

> 20 秒版

今日、高価値な workflow、skills、遊休 agent 算力は孤立したプラットフォームに閉じ込められ、安全に再利用されず、本当の市場価格もついていません。

AgentCoin は機械労働市場のための P2P ツールネットワークです。workflow、skills、compute を発見可能、呼び出し可能、検証可能、価格付け可能な生産資産へ変え、機密の know-how を漏らさずに賃貸・売買・精算できるようにします。

私たちの目標は、機械労働をエネルギーや帯域のように自由に流通させ、開かれた市場へ接続することです。
